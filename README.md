## Description
Application for keeping track of a caffeine drinker's favorite drinks. A user can input a new caffeinated beverage and include ratings of each beverage. Once a user logs in, they can view a list of beverages they have tried and view (as well as update) where that beverage falls in the userâ€™s personal ranking.
